# KPFL

Monorepo structure:

- `backend` - current Spring Boot backend
- `frontend-admin` - admin web frontend
- `ios-user-application` - iOS user app

## Getting Started

Backend was moved from the repository root into [backend](./backend).

Windows:

```bat
cd backend
mvnw.cmd spring-boot:run
```

Linux/macOS:

```bash
cd backend
./mvnw spring-boot:run
```
